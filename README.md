# Data-Professional-Survey-Dashboard

## Project Overview
This project involves a comprehensive analysis of survey data collected from over 600 data professionals globally. Using Power BI, I transformed raw survey data into an interactive dashboard to visualize key industry trends, salary distributions, and career satisfaction.

## Dashboard Preview
![Data Professional Dashboard](Power%20Bi%20Dashboard.png)

## Key Steps Taken:
* **Data Cleaning:** Used Power Query to handle messy "Other" entries, split columns for salary ranges, and standardized country names.
* **Data Transformation:** Created custom DAX measures to calculate average salaries and survey participation counts.
* **Dashboard Design:** Implemented gauges for "Happiness," tree maps for geographic distribution, and stacked bar charts for programming language popularity.

## Key Insights:
* **Salary Trends:** Data Scientists earn the highest average salary ($94k), followed by Data Engineers.
* **Language Popularity:** Python is the dominant language, with R and Other languages trailing significantly behind.
* **Satisfaction:** Data professionals are generally more satisfied with their Work-Life Balance than their actual Salary.

## Tools Used
* **Power BI** (Power Query, DAX, Data Visualization)
* **Data Source:** Survey data provided by Alex The Analyst.
